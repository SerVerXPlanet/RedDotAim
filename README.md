### How to compile

```cmd
- git clone https://github.com/SerVerXPlanet/RedDotAim.git
- cd RedDotAim
- %windir%\Microsoft.NET\Framework\v4.0.30319\MSBuild.exe RedDotAim.sln /property:Configuration=Release
- cd RedDotAim\bin\Release
```
